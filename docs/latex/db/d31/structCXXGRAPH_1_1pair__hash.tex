\doxysubsection{Detailed Description}
The C++ Standard doesn\textquotesingle{}t provide a hash for std\+::pair type, which is required when using std\+::pair as a key in std\+::unordered\+\_\+map. So, hash function for pair needs to be provided. Hash for a pair is calculated by calcuating hash for individual elements and then combining. For combining, an implmentation similar to boost hash\+\_\+combine is implemented.
