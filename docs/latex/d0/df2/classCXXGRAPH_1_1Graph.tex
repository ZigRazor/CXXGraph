Mathematical definition of the problem\+:

Let G be the set of nodes in a graph and n be a given node in that set. Let C be the non-\/strict subset of G containing both n and all nodes reachable from n, and let C\textquotesingle{} be its complement. There\textquotesingle{}s a third set M, which is the non-\/strict subset of C containing all nodes that are reachable from any node in C\textquotesingle{}. The problem consists of finding all nodes that belong to C but not to M.
